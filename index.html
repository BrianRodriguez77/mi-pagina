Pagina web



Que es HTML

HTML significa "HyperText Markup Language" lo que significa lenguaje de marco hipertexto, basicamnete es un lenguaje estrcuturado y fundamental para crear y estructurar el contenido de las páginas web, definiendo todas las estructuras y significados de los elementos como texto, imagenes y enlaces. HTML, se compone de un encabezado, cuerpo y pie de página, el cual se utiliza en las paginas web más utilizadas, además d eesto permite tener un desarrollo mejor, en diferentes áreas de trabajo.

Este lenguaje marcado se compone por las siguientes estructuras:
  • Estructura del contenido: HTML es utilizado para organizar el contenido de las paginas web como texto, imagenes, videos y enlaces. La utilización de etiquetas HTML indican los navegadores web y el contenido a mostrar.
  • Lenguaje marcado: No es un lengujae de programación, por lo cual este utiliza "etiquetas" para poder definir la estructura y el significado del contenido de la página.
  • Fundamento de la web: HTML es la base de todas las páginas web. Sin la utilización de HTML los navegadores no sabrían como mostrar el contenido de una forma adecuada o el contenido del cual se este hablando


Que es CSS

CSS significa "Cascading Style Sheets" lo cual traduce a 'Hojas de estilo en Cascada', es un lenguaje el cual se enfonca en el diseño grafico que se utiliza para controlar y definir las presentaciones de un documento HTML O XML. Esto quiere decir que CSS permite darle una mejor apariencia y estilo a nuestra página web.

  • Contol de diseño visual: CSS permite controlar una amplia gama de aspectos de una página web, incluyendo:
    1. Colores y fondos
    2. Tipografia (fuentes, tamaños, estilos)
    3. Margenes, rellenos y bordes
    4. Diseño y disposición de elementos
    5. Animaciones y transiciones

  • Beneficios de usar CSS:
    1. Consistencia: CSS permite mantener un diseño coherente en todo sitio web.
    2. Eficencia:Separa el estilo del contenido facilita la actualización y el mantenimiento de las páginas web.
    3. FlexibilidadCSS ofrece un alto grado de control sobre el diseño visual de una página web.
    4. Accesibilidad:El uso adecuado de CSS puede mejorar la accesibilidad de un sitio web para personas con discapacidad.


Que es Visual Studio Code

Visual Studio Code (VS Code) es un editor de código fuente desarrollado por Microsoft. A diferencia de un Entorno de Desarrollo Integrado (IDE) completo, VS Code es ligero y rápido, pero altamente personalizable y extensible. Esto lo convierte en una herramienta muy popular entre desarrolladores de diversos lenguajes de programación.

    1. Editor de código fuente:Su función principal es permitir a los desarrolladores escribir y editar el código fuente de aplicaciones y sitios web.
    2. Ligero y rápido:A diferencia de otros, VS Code tiene una memoria pequeña lo cual hace ejecutar de manera eficiente y lo hace adecuado para una amplia gama de computadores.
    3. Multiplataforma:VS Code se encuentra disponible para distintas platafomras como Windows, macOS y Linux, lo que lo convierte en una herramienta muy versatil para desarrollar trabajos en diferentes sistemas opertativos.


Que son etiquetas HTML

Las etiquetas HTML son la base de este lenguaje. Estas se utilizan para estructurar el contenido de las páginas web y decirle a los diferentes tipos de navegadores como deben de mostrar el dicho contenido, así mismo el uso de etiquetas de HTML, sirven para mostrar diferentes funciones y apartados para que los navegadores puedan darle un gran contenido visual al usuario que este viendo dicha página.

  • Las estiquetas HTML son "marcadores" qpermiten definir la estructuraa y el significado de los distintos elementos de una página web.

  • Le dan las respectivas indicaciones al navegador sobre qué tipo de contenido se esta mostrando, como el encabezado, parrafos, imagenes, enlaces, etc.

  • Las etiquetas HTML se describen y se utilizan entre corchetes angulares (< >).

  • Las etiquetas HTML organizan el contenido en una jerarquía lógica, lo que facilita la comprensión y la navegación de la página web.

  • Las etiquetas proporcionan información sobre el contenido, lo que ayuda a los navegadores, motores de busqueda y tecnologías asistentes a interpretar la página web correctamente.


Etiquetas más usadas en html

Dentro de las etiquetas más usadas en HTML son las siguientes:

    Estructura básica de metados:
  • <html>: Define la raíz de un documento HTML

  • <head>: Contiene metadatos sobre el documento HTML.

  • <title>:Define el título del documento que se muestra en la barra de título del navegador.

  • <body>: Define el cuerpo del documento, que contiene el contenido visible de la página.

  • <meta>: Define metados sobre un documento HTML, como la codificación de caracteres y la descripción de la página.

  • <link>: Define la relación entre el documento actual y un recurso externo, como una hoja de estilo CSS.

  • <script>: Se utiliza para insertar scripts del lado del cliente (JavaScript).

    • Contenido de texto:
  • <h1> a <h6>: Define encabezados (titulos) y de diferentes niveles (tamaños).

  • <p>: Define un párrafo o la escritura de este mismo.

  • <a>: Define un hipervínculo.

  • <strong>: Define un texto importante o resaltado.

  • <em>: Define un texto enfatizado.

  • <span>: Es un contenedor de línea genérico para contenido en línea.

  • <br>: Inserta un salto de línea simple.

  • <ol>: Define una lista ordenada.

  • <ul>: Define una lista desordenada.

  • <li>: Define un elemento de lista.

    • Contenido multimedia:
  • <img>: Define una imagen.

  • <video>: Define un video.

  • <audio>: Define un sonido

    • Formularios:
  • <form>: Define un formulario HTML para la entrada del usuario.

  • <input>: Define un control de entrada.

  • <button>: Define un botón ciclable.

  • <label>: Define una etiqueta para un elemento <input>.

  • <select>: Define una lista desplegable.

  • <textarea>: Define un control de entrada de texto multilínea.

    • Contenedores y diseño:
  • <div>: Define una sección o división en un documento HTML.

  • <table>: Define una tabla.

  • <tr>: Define una fila en una tabla.

  • <td>: Define una celda en una tabla.
